# Pizza Exercise

## How to run locally

### Client

- clone repo
```bash
git clone https://github.com/benbousquet/pizza && cd pizza
```
- navigate to client dir
```bash
cd client
```
- install deps and run development server
```bash
npm install && npm run dev
```

### Api

- clone repo
```bash
git clone https://github.com/benbousquet/pizza && cd pizza
```
- navigate to api dir
```bash
cd api
```
- install deps and run development server
```bash
bun install && bun run dev
```
